Description lists are used to display terms and their related description and are made up of two parts: terms and description. Description lists are often used for lists of terms and their definitions. Description lists must use the dl element and can be styled through the .type--dl class.

Terms must use the dt element and can be styled through the .type--dt class.

Descriptions must use the dd element and can be styled through the .type--dd class.

Both ordered and unordered lists share the same styling, making it easy to style. The list wrapper must use either the ul element (for unordered lists) or the ol element (for ordered lists) and can be styled through the .type--list class. List items must use the the li element and can be styled through the .type--list-item class.

Quotes are meant to emphasize extended quotations, be it from the current source or an external source, optionally with some form of citation. They must use the blockquote element and can be styled through the .type--quote class.

Info: This is a message, used to convey non-critical information to a user.

Note: This is a message--note, used to convey important, but not critical, information to a user.

Warning: This is a message--warning, used to convey critical information to a user.

Tip: This is a message--tip, used to convey helpful but not required information to a user.

Messages are used as call-outs to direct readers users to additional, often important, information. They come in four flavors, info, note, warning, and tip, with info being the default. The note version should be used when there's something the user should pay attention to but is not critical. The warning version should be used when there's something critical the user must know. The tip version should be used when there's additional helpful information for the user. If none of the above options make sense, the standard info version should be used.

Messages usually use the aside element and can be styled by using the .message class as well as optionally one of the modifiers .message--note, .message--warning, .message--tip
